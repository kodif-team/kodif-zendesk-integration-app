# Kodif Zendesk Integration App

This is a minimal Zendesk Marketplace integration app package for Kodif Support Automation.

This package is intended for a Zendesk Marketing-only / Integration App submission. It does not include a Zendesk sidebar UI, ZAF iframe app, backend service code, OAuth secrets, or customer automation logic.

## Package Contents

- `manifest.json` — Zendesk app metadata
- `translations/en.json` — Marketplace listing text
- `assets/logo.png` — app logo asset
- `assets/logo-small.png` — small logo asset
